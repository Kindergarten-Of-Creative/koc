# Kindergarten of Creativity – Donation Website (Gaza)

A modern donation platform built to support the **Kindergarten of Creativity in Gaza**, enabling secure USDT donations and hosted payment gateways for global contributors.

---

## 🌐 Project Overview

This website allows donors to:
- Contribute using USDT (multiple networks)
- Use secure hosted payment gateways (card, Apple Pay, Google Pay)
- Access transparent donation information
- Support educational needs for children

All payments are processed through trusted third-party providers. The website does **not store or process sensitive financial data**.
